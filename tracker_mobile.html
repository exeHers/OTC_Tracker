Auto Track tutorial

Step-by-step setup and usage instructions.

1. The APK is only the tracker. Pocket Option must run in a separate browser tab or on your PC.
2. In More tab, enter your relay URL, user key, and token if you use one. Save settings first.
3. Tap Get relay userscript. In the APK it opens a copy window; in a browser it downloads a `.user.js` file.
4. Install that script in Tampermonkey or another userscript manager on the browser that opens Pocket Option.
5. Open Pocket Option, keep the closed deals list visible, then turn Auto Track ON in the APK and tap Sync now.
6. Only trades that close after the script is installed and tracking is on will be collected.

Auto Bot tutorial

Execution bridge setup (no direct Pocket Option API).

1. Save Easy settings (asset, amount, interval).
2. Configure relay in More tab using same user key for bot + executor.
3. Download Bot Executor userscript from this Bot screen.
4. Install userscript in Tampermonkey where Pocket Option runs.
5. Tap Apply bot settings, then start the bot. Live orders need the executor userscript on PO with a public HTTPS relay URL.
